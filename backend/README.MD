### intall dependecies
  `npm install knex`
  `npm install sqlite3`

### Help com os comandos disponiveis
  `npx knex`

### Criar migration
   `npx knex migrate:make < name migration >`

### Subir migations knex sqlite
   `npx knex migrate:latest`
